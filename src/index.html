仪表盘

👋 欢迎使用万遥客户端
🔌
串口助手
连接串口设备，调试通信数据。支持自定义波特率、十六进制收发、时间戳等功能。
🌐
网络助手
TCP/UDP 网络调试工具。支持客户端和服务器模式，方便测试网络通信。
💾
固件烧录
支持 STC/STM32/ESP32 等芯片的固件烧录，一键烧录、验证、擦除。
📈
数据可视化
实时波形显示，支持多通道数据、手动输入和模拟数据，可导出 CSV/JSON。
🧪
自动化测试
创建测试用例，批量执行并生成测试报告。支持文本/JSON/CSV 格式导出。
⚙️
设置
切换深色/浅色主题，检查 Python 环境（可选），查看版本信息。
